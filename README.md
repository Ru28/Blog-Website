# Blog Website

## Setup
### Create Virtual Environment 
```
pip install virtualenv
```
```
virtualenv env
```
### Activate Virtual Environment (Windows)
```
env\Scripts\activate
```

## Installation
### Install all dependencies using requirements.txt
```
pip install -r requirements.txt
```
